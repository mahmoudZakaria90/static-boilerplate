# Static-starterKit
 Static generator Pug to HTML 

 ### Pug doc
https://pugjs.org/


### clone
`git clone git@github.com:mahmoudZakaria90/static-starterKit.git`

### install
`yarn` or `npm i`

### dev
`yarn dev` or `gulp dev`

### build
`yarn build` or `gulp build`

### compile All .pugs to .html
`yarn pug`

Enjoy :heart:
